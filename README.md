TubeWebsite Readme

This repository contains the code for a simple tube website that fetches video content from an external API and displays it on the front end. Below is a brief overview of the project structure, functionalities, and how to get started.
Project Structure
    index.html: The main HTML file containing the structure of the tube website.
    pherotube.js: JavaScript file handling the API requests and dynamic content rendering.
    Logo.png: Logo image file used in the header.
    Icon.png: Icon image file displayed when no content is found.
    blog.html: HTML file for the blog page.
    README.md: Documentation file you are currently reading.
Getting Started

    Clone the repository to your local machine:
    git clone https://github.com/your-username/tubewebsite.git
Open the index.html file in your web browser.

Explore the tube website, and use the "Sort by view" button to sort videos by views.

Features

    Dynamic Content: The website dynamically fetches and displays video content from an external API.
    Category Buttons: Categories are displayed as buttons, and clicking on a button loads videos of that category.
    Sort by View: The "Sort by view" button allows users to sort videos based on their view count.
    Blog Link: Clicking the "Blog" button redirects users to the blog.html page.

API Usage

    The website uses the Programming Hero API to fetch video content.

Customization

    You can customize the website by modifying the HTML, CSS, and JavaScript files based on your preferences.
    Update the API endpoints or integrate with a different API as needed.

Dependencies

    tailwind CSS: Used for styling the website. The stylesheet is linked from a CDN.
    fontAwesome: Icons are not currently used, but the library is included and can be added for additional features.
